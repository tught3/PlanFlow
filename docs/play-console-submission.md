# PlanFlow Play Console 제출 준비 문서

이 문서는 Play Console 개발자 계정 인증이 끝난 뒤 앱 생성, 앱 콘텐츠, 데이터 보안, 내부 테스트 등록 화면에 옮겨 적을 값을 모아둔 문서입니다.

- 기본 언어: `한국어(대한민국)`
- 앱 또는 게임: `앱`
- 무료/유료: `무료`

- 현재 내부 테스트 versionCode: `18`

## 스토어 등록 문구 초안

### 짧은 설명

말하면 일정이 됩니다. AI 음성으로 일정 등록, 캘린더 연동, 출발 알림까지 자동으로.

### 전체 설명

PlanFlow는 음성으로 일정을 입력하고, 약속 장소까지 이동 시간을 계산해 출발 시각까지 알려주는 AI 일정 관리 앱입니다.

"내일 오후 2시 강남 미팅"처럼 말하면 날짜, 시간, 장소, 제목을 자동으로 파악해 일정으로 저장합니다. Google 캘린더·Naver 캘린더와 연동하고, 출발 알림·아침저녁 브리핑까지 자동으로 챙겨줍니다.

주요 기능:

- AI 음성 일정 입력 (말 한마디로 등록)
- 이동시간 자동 계산 · 출발 알림
- Google Calendar, Naver CalDAV, 휴대폰 내부 캘린더 연동
- 아침·저녁 브리핑 (AI 음성 요약)
- 스마트 준비 알림 (역산 알림)
- Android 홈 위젯 (잠금 해제 없이 바로 입력)
- 일정 백업 및 복원

PlanFlow는 음성 파일을 외부 서버로 전송하지 않습니다. 기기에서 변환된 텍스트만 일정 파싱과 브리핑 생성에 사용합니다.

## 내부 테스트 릴리즈 노트

### 한국어

PlanFlow 1.1.0 내부 테스트 버전입니다.

- 음성 일정 입력 및 일정 저장
- Google/Naver/휴대폰 내부 캘린더 동기화
- 장소 기반 출발 알림
- 모닝/이브닝 브리핑
- 중요 알림 권한 상태 표시
- Android 홈 위젯
- 백업 및 복원

이번 1차 내부 테스트에는 KakaoTalk/SMS 자동 감지 기능이 포함되지 않습니다. 해당 기능은 추후 별도 권한 안내와 함께 검증 후 제공할 예정입니다.

테스트 중에는 로그인, 일정 저장, 캘린더 연결, 알림 권한, 출발 알림, 백업 기능을 중점적으로 확인해 주세요.

## 내부 테스트 AAB 자동 생성

내부 테스트용 AAB를 만들 때는 루트에서 아래 중 하나를 실행하면 됩니다.

```powershell
.\deploy-planflow.bat
```

또는

```powershell
.\scripts\build-internal-aab.ps1
```

이 스크립트는 `pubspec.yaml`의 `version`에서 빌드 번호만 1 올린 뒤,
`flutter analyze` → 주요 테스트 → `release appbundle` 순서로 실행하고,
마지막에 `build/app/outputs/bundle/release/app-release.aab` 전체 경로를 출력합니다.

## 앱 액세스

- 앱 전체 기능은 로그인이 필요합니다.
- Play 검토용 테스트 계정을 만들 경우 별도 문서나 Play Console 앱 액세스 항목에 이메일/비밀번호를 입력합니다.
- OAuth 제공자 Google/Naver/Kakao는 외부 콘솔 설정이 완료되어야 정상 동작합니다.

- Naver Calendar/CalDAV: 사용자가 연결한 경우 일정 읽기/쓰기

## 권한 설명 문구

### 마이크

음성으로 일정을 입력하기 위해 사용합니다. 음성 파일은 외부 서버로 전송하지 않습니다.

### 알림

일정 알림, 준비 알림, 출발 알림, 모닝/이브닝 브리핑 알림을 표시하기 위해 사용합니다.

### 정확한 알람

중요 일정과 출발 알림을 더 정확한 시간에 울리기 위해 사용합니다. Android 정책상 기기 설정에 따라 제한될 수 있습니다.

### 위치

장소 선택, 이동시간 계산, 출발 알림을 위해 사용합니다.

### 캘린더

휴대폰 내부 캘린더 일정을 가져오거나 PlanFlow 일정을 내보내기 위해 사용합니다.
