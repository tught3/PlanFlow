PlanFlow 계정 및 데이터 삭제 안내

이 문서는 FluxStudio가 제공하는 PlanFlow 앱의 계정 삭제와 사용자 데이터 삭제 요청 방법을 안내합니다.

1. 계정 삭제 요청 방법

PlanFlow 계정 삭제를 원하시면 아래 정보를 포함해 officialfluxstudio.kr@gmail.com으로 요청해 주세요.

  • 메일 제목: PlanFlow 계정 삭제 요청
  • PlanFlow에 로그인한 이메일 주소
  • 삭제 요청 범위: 계정 전체 삭제

요청을 확인한 뒤 본인 확인이 필요한 경우 추가 확인을 요청할 수 있으며, 확인이 완료되면 계정과 앱 서비스 제공에 필요한 사용자 데이터를 삭제합니다.

2. 삭제되는 데이터

계정 전체 삭제 시 다음 PlanFlow 데이터가 삭제됩니다.

  • 로그인 계정 정보와 앱 프로필 정보
  • 사용자가 생성한 일정, 알림, 준비 항목, 위치 기록
  • 음성 입력에서 변환된 텍스트 로그와 일정 분석 기록
  • 캘린더 연동 상태, 네이버 CalDAV 연결 정보, 앱 설정
  • 앱 백업 및 복원 데이터
  • 앱 내부 문제 신고와 피드백 내역 중 계정과 연결된 데이터

3. 보관될 수 있는 데이터와 보관 기간

법적 의무, 보안 감사, 부정 이용 방지, 고객 지원 이력 확인을 위해 일부 기록은 필요한 범위에서 최대 90일간 보관될 수 있습니다. 보관 기간이 끝나면 삭제하거나 개인을 식별할 수 없도록 처리합니다.

4. 계정 삭제 없이 일부 데이터 삭제 요청

계정은 유지하고 일부 데이터만 삭제하고 싶다면 앱 안에서 일정을 직접 삭제하거나, 아래 정보를 포함해 같은 이메일 주소로 요청해 주세요.

  • 메일 제목: PlanFlow 데이터 삭제 요청
  • PlanFlow에 로그인한 이메일 주소
  • 삭제하려는 데이터 유형: 일정, 백업, 음성 로그, 캘린더 연동 정보, 문제 신고 등
  • 삭제 대상 기간 또는 구체적인 설명

일부 데이터 삭제는 앱 기능 유지에 필요한 최소 계정 정보는 남긴 상태로 처리됩니다.
