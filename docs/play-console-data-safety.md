# PlanFlow Play Console 데이터 보안 답변표

이 문서는 Google Play Console의 데이터 보안/앱 콘텐츠 입력을 빠르게 반복할 수 있도록 만든 제출 보조표입니다. 실제 입력 전에는 현재 앱 기능과 공개 웹페이지가 같은지 마지막으로 확인합니다.

## 기본 답변

| 질문 | 답변 |
| --- | --- |
| 앱에서 필수 사용자 데이터 유형을 수집하거나 공유하나요? | 예 |
| 앱에서 수집하는 모든 사용자 데이터를 암호화하여 전송하나요? | 예 |
| 지원하는 계정 생성 방법 | 사용자 이름 및 비밀번호, OAuth |
| 계정 URL 삭제 | `https://fluxstudio.co.kr/account-deletion` |
| 계정 삭제 없이 일부/전체 데이터 삭제 요청 방법 제공 | 예 |
| 데이터 URL 삭제 | `https://fluxstudio.co.kr/data-deletion` 또는 `https://fluxstudio.co.kr/account-deletion#partial-data-deletion` |
| 개인정보처리방침 URL | `https://fluxstudio.co.kr/privacy` |

| 항목 | 수집됨 여부 | 공유됨 여부 | 임시처리여부 | 데이터수집 필수여부 | 수집되는 이유 | 공유되는 이유 |
| --- | --- | --- | --- | --- | --- | --- |
| 이름 | 예 | 아니요 | 아니요 | 필수 | 앱 기능, 계정 관리 | 해당 없음 |
| 이메일 주소 | 예 | 아니요 | 아니요 | 필수 | 앱 기능, 계정 관리 | 해당 없음 |
| 사용자 ID | 예 | 아니요 | 아니요 | 필수 | 앱 기능, 계정 관리 | 해당 없음 |
| 대략적인 위치 | 예 | 아니요 | 아니요 | 선택 | 앱 기능 | 해당 없음 |
| 정확한 위치 | 예 | 아니요 | 아니요 | 선택 | 앱 기능 | 해당 없음 |
| 캘린더 일정 | 예 | 예 | 아니요 | 선택 | 앱 기능 | 앱 기능 |
| 앱 상호작용 | 예 | 아니요 | 아니요 | 필수 | 애널리틱스 | 해당 없음 |
| 기타 사용자 제작 콘텐츠 | 예 | 아니요 | 아니요 | 필수 | 앱 기능 | 해당 없음 |
| 비정상 종료 로그 | 예 | 아니요 | 아니요 | 필수 | 애널리틱스 | 해당 없음 |
| 진단 | 예 | 아니요 | 아니요 | 필수 | 앱 기능, 애널리틱스 | 해당 없음 |
| 기타 앱 성능 데이터 | 예 | 아니요 | 아니요 | 필수 | 애널리틱스 | 해당 없음 |
| 기기 또는 기타 ID | 예 | 아니요 | 아니요 | 필수 | 앱 기능, 애널리틱스 | 해당 없음 |

## 선택하지 않는 데이터 유형

- 주소, 전화번호, 인종 및 민족, 정치적 또는 종교적 신념, 성적 지향, 기타 개인 정보
- 사용자 결제 정보, 구매 내역, 신용 점수, 기타 금융 정보
- 건강 정보, 피트니스 정보
- 이메일, SMS 또는 MMS, 기타 인앱 메시지
- 사진, 동영상
- 음성 또는 사운드 녹음본, 음악 파일, 기타 오디오 파일
- 파일 및 문서
- 연락처
- 앱 내 검색 기록, 설치된 앱, 기타 작업
- 웹 방문 기록

## 앱 콘텐츠/콘텐츠 등급 참고 답변

| 질문 | 답변 |
| --- | --- |
| 앱이 음성 통신, 텍스트, 이미지, 오디오 공유를 통해 다른 사용자와 상호 작용하거나 콘텐츠를 교환하도록 허용하나요? | 아니요 |
| 최초 다운로드 시 포함되지 않지만 앱에서 접근 가능한 콘텐츠를 제공하거나 홍보하나요? | 아니요 |
| 사용자의 현재 정확한 실제 위치를 다른 사용자와 공유하나요? | 아니요 |
| 디지털 상품 구매를 허용하나요? | 아니요 |
| 현금 보상, 기프트 카드, P2E, 암호화폐/NFT 보상이 포함되어 있나요? | 아니요 |
| 웹브라우저 또는 검색 엔진 앱인가요? | 아니요 |
| 앱이 주로 뉴스 또는 교육용 제품인가요? | 아니요 |
| 권장 타겟 연령대 | 만 18세 이상 |

## 앱 액세스 권한 안내 문구

```text
앱의 주요 기능은 로그인 후 사용할 수 있습니다.
검토자는 아래 테스트 계정으로 이메일 로그인을 진행할 수 있습니다.
로그인 후 음성 일정 입력, 일정 확인, 캘린더 동기화 설정, 알림 설정 기능을 확인할 수 있습니다.
```

사용자 이름과 비밀번호 외에 추가 접근 정보는 필요하지 않은 경우 Play Console의 "앱에 액세스하는 데 다른 정보가 필요하지 않음"을 선택합니다.
